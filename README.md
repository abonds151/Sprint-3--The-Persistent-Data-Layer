Milestone 3: Database Integration & Persistence
Replace temporary server variables with a persistent relational database using Prisma and PostgreSQL.

 

Core Objectives
Schema Modeling: Define your project's data architecture in Prisma.

Database Provisioning: Connect to a hosted PostgreSQL instance (e.g., Neon).

ORM Operations: Use the Prisma Client to perform CRUD operations within your API routes.

Data Integrity: Implement server-side validation to ensure data quality.

The project includes user and item models and one-to-many relationship between users and items.
